## üöÄ Quick Start

1. **Clone the repo**  
   ```bash
   git clone <repository-url> LightBlueAngular
   cd LightBlueAngular
   ```
2. **Install dependencies**  
   ```bash
   yarn install  # Do NOT use npm!
   ```
3. **Run the app**  
   ```bash
   yarn start
   ```
   Navigate to http://localhost:3000/.

4. **Backend Support**  
   ```bash
   yarn run start:backend
   ```

5. **Build for production**  
   ```bash
   yarn build --prod
   ```

> **Pro Tip:** If something breaks, it's probably your fault. üòú
